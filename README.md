# Modern Resume React Site

A complete Vite + React resume website with a modern responsive layout, editable content, and production build scripts.

## Run locally

```bash
npm install
npm run dev
```

## Edit the resume

Update `src/resumeData.js` with your own name, role, contact details, work history, projects, skills, and education.

Replace `public/profile-mark.svg` with your own portrait or brand image if you want a more personal hero visual.

## Build

```bash
npm run build
```
